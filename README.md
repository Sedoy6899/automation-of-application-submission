## 🎯 О проекте

Тренеры отправляли заявки на почту, и организаторам приходилось вручную переносить данные в Excel. Это занимало много времени и приводило к ошибкам.

**Решение:** Создан веб-сервис с интуитивным интерфейсом, где тренеры самостоятельно заполняют формы. Данные автоматически попадают в Google Таблицу и распределяются по соответствующим листам в зависимости от программы, дивизиона и категории.

## ✨ Функциональность

- **4 типа программ**: Индивидуальная, Групповое упражнение, Двойки и тройки, ОФП
- **Добавление нескольких гимнасток** в одной заявке
- **Автоматическое распределение** по листам Google Таблицы:
  - Индивидуальные программы → по дивизионам (30 листов)
  - Групповые упражнения → "Групповые упражнения"
  - Двойки → "Двойки"
  - Тройки → "Тройки"
  - ОФП → по годам обучения
- **Проверка данных**:
  - Год рождения → только 4 цифры
  - Город → только буквы
- **Предпросмотр заявки** перед отправкой
- **Ограничения**:
  - Групповые: максимум 6 гимнасток
  - Двойки: максимум 2
  - Тройки: максимум 3
- **Автоматическая нумерация** записей
- **Защита от конфликтов** при одновременной отправке (LockService)

## 🛠 Технологии

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Google Apps Script
- **База данных**: Google Sheets (выступает в роли базы данных)
- **Хостинг сайта**: GitHub Pages / Netlify

## 📊 Структура листов Google Таблицы

### Индивидуальная программа
30 листов формата: `{дивизион} дивизион {год}`
- 1 дивизион 2020 и младше, 1 дивизион 2019, ... 1 дивизион 2011 и старше
- 2 дивизион 2020 и младше, ... 3 дивизион 2011 и старше

### Групповое упражнение
- `Групповые упражнения`

### Двойки и тройки
- `Двойки`
- `Тройки`

### ОФП
- `ОФП 1 год`
- `ОФП 2-3 год`

## 🔧 Установка и настройка

### Шаг 1: Создайте Google Таблицу
1. Создайте новую Google Таблицу
2. Скопируйте ID из URL (часть между `/d/` и `/edit`)

### Шаг 2: Настройте Google Apps Script
1. Откройте **Расширения → Apps Script**
2. Вставьте код из файла `Code.gs`
3. Замените `SHEET_ID` на ID вашей таблицы
4. Запустите функцию `createAllSheets()` для создания всех листов
5. Опубликуйте скрипт как веб-приложение:
   - Нажмите **"Развернуть" → "Новое развертывание"**
   - Выберите тип **"Веб-приложение"**
   - Доступ: **"Все"**
   - Скопируйте полученный URL

### Шаг 3: Настройте сайт
1. В файле `index.html` найдите строку:
   ```javascript
   const FORM_URL = 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE';
   
2. Замените `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE` на URL вашего веб-приложения

### Шаг 4: Опубликуйте сайт
Вариант A: GitHub Pages
  Создайте репозиторий username.github.io

  Загрузите index.html

  В настройках репозитория включите GitHub Pages

Вариант B: Netlify
  Перетащите index.html на Netlify

  Получите ссылку

## Использование
- Тренер переходит по ссылке на сайт
- Выбирает нужную вкладку
- Заполняет общую информацию
- Добавляет гимнасток (кнопка "+ Добавить гимнастку")
- Проверяет данные в предпросмотре
- Подтверждает отправку
- Данные автоматически появляются в Google Таблице
